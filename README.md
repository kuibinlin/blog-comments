# blog-comments

This repository hosts the reader comments for my blog,
[linsnotes.com](https://linsnotes.com), powered by
[giscus](https://giscus.app) (built on GitHub Discussions).

Each blog post maps to a discussion in this repo. When you leave a
comment under a post, it's stored here as a reply in the **Comments**
discussion category.

There's no code here — just Discussions. To comment, visit a post on
the blog and sign in with your GitHub account.
